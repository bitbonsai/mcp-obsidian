# LLM.txt - MCP-Obsidian

## Project Description
MCP-Obsidian is an open-source Model Context Protocol (MCP) server that provides AI assistants with safe, intelligent access to Obsidian vaults. It enables natural language interaction with personal knowledge bases through secure, structured protocols.

## Key Features
- **Secure Vault Access**: AI-safe interface to Obsidian knowledge bases
- **Natural Language Queries**: Search and interact with notes using conversational language
- **Note Management**: Create, edit, and organize notes through AI assistants
- **Link Analysis**: Understand and navigate knowledge connections
- **Template Support**: Automated note creation with predefined structures
- **Tag Management**: Organize and filter content by topics
- **Attachment Handling**: Process and reference embedded media files

## Technical Stack
- **Protocol**: Model Context Protocol (MCP)
- **Runtime**: Node.js/Bun
- **Language**: TypeScript
- **Platform**: Cross-platform (macOS, Windows, Linux)
- **Integration**: Claude Desktop, ChatGPT, and other MCP-compatible AI platforms

## Use Cases
- **Personal Knowledge Management**: AI-powered note organization and retrieval
- **Research Assistance**: Intelligent analysis of research notes and references
- **Writing Support**: Context-aware content generation from existing knowledge
- **Learning**: Interactive exploration of interconnected concepts
- **Project Management**: AI-assisted task and project note coordination
